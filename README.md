# ICHgram Frontend

ICHgram is a full-stack Instagram-like social media application built with React.

Users can create accounts, log in, manage profiles, publish posts, like and comment on posts, follow other users, search profiles, exchange messages and receive notifications.

## Features

- User registration and login
- JWT authentication
- User profile management
- Avatar upload
- Create, edit and delete posts
- Image upload with Cloudinary
- Likes and comments
- Delete own comments
- Follow and unfollow users
- User search
- Home feed
- Explore page
- Direct messages
- Real-time communication with Socket.io
- Notifications for likes, comments and follows
- Responsive layout
- Protected routes

## Technologies

- React
- JavaScript
- Vite
- React Router
- Axios
- Socket.io Client
- Lucide React
- CSS Modules and CSS
- Vercel
